About

SpaceX Launch Tracker is an open source web application for exploring data about upcoming and past SpaceX launches. It was created by Emil Sadek using Flask and Bootstrap, with data from the SpaceX-API. If you are interested in the source code or contributing to this project, check out the GitHub repository. Special thanks to everyone at SpaceX for their incredible work, as well as Jake Meyer for creating and maintaining the SpaceX-API.

Disclaimer: This website and its creator are not affiliated with Space Exploration Technologies Corp. (SpaceX).
